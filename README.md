# e-commerce

###  The back-end of an e-commerce site.

 ------------------

## TECHNOLOGIES USED:

- Node.js
- MySQL
- Javascript 

 ------------------

## DEPENDENCIES:

- dotenv
- Express
- MySQL2
- Node
- Sequelize
- Nodemon

 ------------------

## INSTALLATION:

- In integrated terminal:

        npm i mysql2
        npm i node
        npm i dotenv
        npm i sequelize
        npm i express

- In MySQL terminal: 

        -u root -p 
        source db/schema.sql

- In integrated terminal:
            
        node server.js
        npm run seed
